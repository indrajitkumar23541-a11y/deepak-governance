# 🎓 Deepak Kumar - Academic Portfolio

> A premium, highly responsive personal portfolio highlighting my academic journey in Arts & Humanities.

![Portfolio Banner](./src/assets/hero-premium-dark.png)

## 🌟 About The Project

This is a modern, responsive web application designed to showcase my academic background, subjects of interest (History, Civics, Geography), and my aspirations for the UPSC examination. Built with a focus on elegant design, smooth animations, and a rich user experience.

### ✨ Key Features

- **Dynamic Dark/Light Mode**: Seamlessly switch between a clean light theme and a sleek dark theme.
- **Premium Aesthetics**: Glassmorphism, smooth gradients, and micro-interactions ensure a visually stunning experience.
- **Responsive Design**: Flawless experience across all devices—desktops, tablets, and smartphones.
- **Interactive Sections**: 
  - Dynamic **Subject** and **Note** explorers.
  - Interactive **Timeline** for academic achievements.
  - Interactive **Gallery** with a secure Admin mode for uploading moments.
- **Smooth Animations**: Powered by Framer Motion for scroll reveals, hover effects, and page transitions.

## 🛠️ Technology Stack

- **Framework**: [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Styling**: Vanilla CSS (Modular, with CSS Variables for themes)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [React Icons](https://react-icons.github.io/react-icons/)

## 🚀 Getting Started

To run this project locally, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/indrajitkumar23541-a11y/deepak-governance.git
   ```
2. **Navigate to the directory:**
   ```bash
   cd deepak-governance
   ```
3. **Install dependencies:**
   ```bash
   npm install
   ```
4. **Start the development server:**
   ```bash
   npm run dev
   ```

## 🔒 Admin Mode Access

The **Photo Gallery** section includes a hidden Admin Mode for managing photos.
- **How to access**: Navigate to the Gallery section and **Double-Click** (or **Double-Tap** on mobile) the "Photo Gallery" heading.
- You will be prompted for an admin password to unlock upload and delete capabilities.
